# TikTak Toe Game

_A fun game to play with friends_

### Tech Stacks

- React
- TypeScript
- Vite
- Zustand

### how to install

```bash
git clone https://github.com/thihaswe-work/react-tiktak-toe.git
```

```bash
cd react-tiktak-toe
```

```
npm install
```

```cmd
npm run dev
```

the game will appears on `localhost:3000` of your browser.
